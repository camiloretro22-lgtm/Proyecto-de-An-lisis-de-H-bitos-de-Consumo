# Proyecto-de-An-lisis-de-H-bitos-de-Consumo

Este proyecto consiste en un análisis exploratorio de datos (EDA) sobre el conjunto de datos de Instacart, una plataforma de entrega de comestibles. El objetivo principal es limpiar los datos y analizar el comportamiento de compra de los usuarios para extraer información valiosa sobre cuándo y cómo realizan sus pedidos.
